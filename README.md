# ⭐️ OlympicsTraining
> Проект для участия в московской предпрофессиональной олимпиаде школьников.
> Практика: Информационные технологии (кейс №1) Платформа для подготовки к олимпиадам.
## Глава 1: Установка проекта
1. Скачайте репозиторий с сайта **GitHub** и разархивируйте его
2. Проверьте наличие современной версии **Python** на сервере:

    Откройте **Терминал** и напишите следующую команду
   - Для **Windows**: ``python -V``
   - Для **Linux / MacOS**: ``python3 -V``

    Версия должна быть не меньше **3.7**. В случае если язык
не установлен на устройстве или версия устарела скачайте
актуальную сборку языка по [ссылке](https://www.python.org/downloads/).
3. Установите необходимые для проекта библиотеки:
   
   В **Терминале** откройте папку с проектом и выполните команду
   - Для **Windows**: ``pip install -r requirements.txt``
   - Для **Linux / MacOS**: ``pip3 install -r requirements.txt``
4. Установка проекта завершена, можете приступать к главе с использованием!

## Глава 2: Использование проекта
Для запуска сервера сайта выполните следующую команду в **Терминале**:
- Для **Windows**: ``python server.py``
- Для **Linux / MacOS**: ``python3 server.py``

При необходимости вы можете изменить хост или порт в файле: ``config.py``

Также через сервер можно сделать пользователя администратором командой:
- Для **Windows**: ``python server.py --make-admin``
- Для **Linux / MacOS**: ``python3 server.py --make-admin``

После чего программа запросит у вас логин уже зарегистрированного на сайте пользователя
и сделает его администратором. Только администратор может добавлять свои задания на сайт!

## Глава 3: Добавление задач
Администратор может добавлять задачи вручную, сохранять из JSON файла
или импортировать с сайта Решу ОГЭ / ЕГЭ.
### Сохранить задачу из JSON файла:
Исходный файл должен обязательно содержать следующие поля:
- **title** – Название задачи
- **condition** – Условие задачи
- **answer** – Ответ на задачу
- **subject** – Задача по предмету
- **difficult** – Сложность **[simple, medium, hard]**

### Импортировать задачу с сайта Решу ОГЭ / ЕГЭ:
Выбирайте такие задачи для импорта, где всё условие задачи находится
в одном текстовом блоке и не содержат таблиц или свёрнутых данных. Задачи
в которых есть картинка будут импортированы успешно!

Пример ссылки: https://math-ege.sdamgia.ru/problem?id=27209

Или: https://math-ege.sdamgia.ru/problem?id=26804
